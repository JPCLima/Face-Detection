## How to install?


1. Clone the repo
```
git clone https://github.com/JPCLima/Face-Detection.git
```
2. Create a virtual env
```
python -m venv env
```
3. Install requirements
```
pip install -r requirements.txt
```
4. Run application
```
streamlit run app.py
```
